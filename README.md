# PenalAI Italia

### ⚖️ Assistente Legale per il Diritto Penale Italiano

Prototipo di supporto alle decisioni per avvocati penalisti.
Analizza atti, citazioni e documenti per fornire strategie e riferimenti normativi.

## Deployment su Streamlit Cloud

1. Carica questi file su GitHub.
2. Vai su [streamlit.io/cloud](https://streamlit.io/cloud).
3. Connetti questo repository.
4. In **Settings > Secrets**, aggiungi:
   ```toml
   OPENAI_API_KEY = "la-tua-chiave"
   ```
5. Clicca su **Deploy**.
